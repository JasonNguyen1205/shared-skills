# shared-skills

Reusable skills and workflows for all Jason repositories.

## Skills

### project-cleanup-and-push
- Path: `project-cleanup-and-push/SKILL.md`
- Use when finalizing work for release.
- Covers: assess changes, cleanup, tests, commit/push via SSH, deploy health verification.

### code-review-and-consolidation
- Path: `code-review-and-consolidation/SKILL.md`
- Use when reviewing current project status and reducing technical duplication.
- Covers: duplicate logic integration, obsolete function/feature removal, risk validation.

## Suggested Invocation Phrases
- "Use project-cleanup-and-push to finalize this branch."
- "Use code-review-and-consolidation to merge duplicate logic and remove obsolete features."
